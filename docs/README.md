# 📂 Strategic Documentation (Executive Abstracts)

This directory provides the conceptual framework and research methodology for **Project Earth Lamia**. Full technical whitepapers are restricted to maintain Operational Security (OpSec).

## 📑 [Abstract] Technical Whitepaper: The CI/CD "Dark Zones"
- **Focus:** In-depth analysis of **CVE-2019-18935** (Insecure Deserialization in Telerik UI).
- **Core Discovery:** Identification of a **2,000+ day Remediation Gap** within the Vietnamese educational sector.
- **Key Techniques:** Detailed breakdown of **Reflective Code Loading (T1620)** and memory-resident persistence.
- **Goal:** Providing a blueprint for proactive hunting in legacy web environments.

## 📑 [Abstract] Project Proposal: NLnet Foundation (NGI Zero)
- **Objective:** Scaling the **Malware Analyzer Pro** toolkit to bridge the gap between automated detection and manual forensic analysis.
- **Strategic Impact:** Protecting 250,000+ educational entities through open-source, behavior-based detection rules (YARA).
- **Roadmap:** Integration of automated IoC extraction and relational intelligence schemas (MITRE-mapped).
