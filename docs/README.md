# 📑 Executive Summary: Project Earth Lamia (UNC5174)

## 🎯 The "Remediation Gap" Paradox
This research analyzes the strategic exploitation of key educational infrastructures by **Earth Lamia (UNC5174)**. A critical finding is the **Remediation Gap**, where high-severity vulnerabilities like **CVE-2019-18935** (Telerik RadAsyncUpload) remain unpatched for up to **~2,190 days (6 years)**.

## 🕵️ CI/CD "Dark Zones"
Attackers successfully exploited **CI/CD Dark Zones**—non-typical installation paths generated by automated pipelines—allowing them to bypass traditional IP scanners and WAFs. The investigation deepens into the **Insecure Deserialization** mechanism using the **AssemblyInstaller** gadget to execute remote code (RCE) directly within the **w3wp.exe** process memory.

## 🛡️ The Malware Analyzer Pro Solution
To address these stealthy threats, the **Malware Analyzer Pro** project provides an innovative solution based on **Memory Forensics**. It focuses on detecting **Reflective Code Loading (T1620)** and anomalous heap behaviors in RAM, rather than relying solely on physical file artifacts on disk.
